# v0.1.0 artifact retention

No generated v0.1.0 runtime artifact is present in this Git checkout. The original acceptance
workspace produced browser screenshots and traces, TXT/JSON/CBZ downloads, recovery/deletion
evidence, and helper scripts below the then-local `artifacts/acceptance-v010/` path. They were not
tracked and were not among the published Release assets, so this archive records their provenance
without fabricating or recreating them.

The final wheel, source archive, supplemental report, and checksum manifest remain available from
the immutable [v0.1.0 GitHub Release](https://github.com/taoning0403/lingua-spindle/releases/tag/v0.1.0);
their verified sizes and SHA-256 digests are listed in the version [acceptance index](../README.md).
